# CP & DSA

This repository contains competitive programming and data structures & algorithms solutions organized by platform and contest.

## Structure

- `Codechef/`
  - Solutions for CodeChef contests, organized by contest code directories.
- `Codeforces/`
  - Solutions for Codeforces problems, with filenames referencing contest and problem codes.

## Purpose

- Archive solved problems and practice implementations.
- Keep contest solutions organized for review and learning.
- Maintain a structured library of competitive programming code.

## Usage

- Open the relevant platform folder (`Codechef` or `Codeforces`).
- Each contest or problem is stored in its own file or folder.
- Use this repository as a reference while preparing for contests or studying problem-solving patterns.

## Notes

- Files are primarily C++ source code (`.cpp`).
- Naming conventions include contest/problem identifiers for easy lookup.
- Add new solutions in the appropriate platform folder.

## Contributing

- Add new contest folders or problem files as you solve more challenges.
- Keep filenames descriptive using the contest code and problem title or identifier.
- Update this README if the repository structure changes.
